# AI Document Intelligence Agent

## Overview

This project extracts structured invoice information from PDF and image documents using an AI Agent workflow.

## Features

- Multiple OCR Models
  - EasyOCR
  - RapidOCR
  - docTR

- Agent-Based OCR Selection

- OCR Quality Evaluation

- Prompt Optimization

- Llama 3.2 Integration

- FastAPI Backend

- Streamlit Dashboard

- JSON Output
